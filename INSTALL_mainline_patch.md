# 安裝方式

1. 先備份你原本專案
2. 把 `replace_into_project/` 內所有檔案複製到專案根目錄
3. 保留舊的 `formal_trading_system_v79.py`
4. 直接執行 `formal_trading_system_v80_prebroker_sealed.py`

## 建議先看這些 runtime 輸出
- `runtime/prebroker_seal_layer_status.json`
- `runtime/formal_trading_system_v80_prebroker_sealed_report.json`
- `runtime/live_safety_gate.json`
- `runtime/model_governance_status.json`
