做法：
1. 英文邏輯表名 -> 中文實體表名，統一由 sql_table() 轉換。
2. 保留你前一版 SQL 相容層策略；本包主要是讓 Python 主線不再硬寫英文表名。
3. 這一版沒有大改所有零散舊模組，只先處理主線/常用模組。

建議：
- 先套用「SQL英文table改中文_相容層_patch」
- 再套用本包
- 之後逐步把剩餘零散 SQL 字串也切到中央映射層
